Qui suis-je

Passionné par le code,

Je suis Mezatio Geril, développeur Web & Mobile basé au Cameroun. Mon quotidien, c'est concevoir et livrer des applications robustes, performantes et élégantes.

Mon expertise couvre le back-end Java Spring Boot pour les APIs sécurisées et scalables, Django pour les applications web Python, et Flutter pour des apps mobiles cross-platform qui cartonnent autant sur iOS qu'Android.

Ce qui me motive ? Transformer des idées complexes en produits simples à utiliser, avec un code propre dont je suis fier.
